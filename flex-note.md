# CSS Flex Property Guide

## Basic Properties

The `flex` property is a shorthand for:
- `flex-grow`
- `flex-shrink` 
- `flex-basis`

## Common Values

```css
/* Most used flex values */
flex: 0;        /* flex: 0 1 auto */
flex: 1;        /* flex: 1 1 0% */
flex: auto;     /* flex: 1 1 auto */
flex: none;     /* flex: 0 0 auto */
flex: initial;  /* flex: 0 1 auto */
```
